Tabkey

Setup in 30 seconds

  1. Pin Tabkey from the Chrome toolbar so it is always visible.
  2. Open Tabkey from the toolbar icon or Alt+Shift+K.
  3. Assign numbers 1 to 9 to your most-used tabs.
  4. Press the same number on any normal web page to jump instantly.

Tabkey cannot run on internal browser pages like chrome:// or Chrome Web Store tabs.
